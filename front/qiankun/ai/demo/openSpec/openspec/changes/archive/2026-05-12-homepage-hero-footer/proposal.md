## Why

手机产品需要一张线上名片。官网首页是用户接触品牌的第一触点——一个极简、理性的首屏能在 3 秒内传达产品定位与核心价值，让访客快速理解"这是一款什么样的手机"并采取下一步行动。当前没有官网页面，需要从零构建。

## What Changes

- 新增 Hero 区：全屏首屏，文字驱动，白底黑字，理性技术调性，包含主标题、副标题、双 CTA 按钮
- 新增 Footer 区：一行精简式底栏，包含关键链接、社交媒体图标、版权与法律信息
- 新增 HTML 文件（index.html）和独立 CSS 文件（style.css）
- 页面为纯静态实现，无 JavaScript 框架依赖

## Capabilities

### New Capabilities
- `hero-section`: 首屏 Hero 区，文字驱动布局，包含品牌主标题、理性技术向副标题、双 CTA 按钮、全屏居中排版、响应式适配
- `footer-section`: 一行精简式 Footer，包含链接导航、社交媒体入口、版权与法律声明
